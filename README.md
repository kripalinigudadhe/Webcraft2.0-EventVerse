📌 Overview

EventVerse is a full-stack web application designed to manage events efficiently. It allows users to explore events, register seamlessly, and enables organizers to create and manage events in real time.

The platform focuses on user-friendly UI, secure authentication, and scalable event handling, making it ideal for college events, hackathons, and tech fests.

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/kripalinigudadhe/Webcraft2.0-EventVerse.git
cd Webcraft2.0-EventVerse
2️⃣ Install Dependencies
npm install
3️⃣ Setup Environment Variables

Create a .env file:

PORT=5000
DB_URI=your_database_url
JWT_SECRET=your_secret_key
4️⃣ Run the Project
npm start
